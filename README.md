# Db-Compare
A quick and dirty tool which allows you to query multiple databases in the same view.
## Server
a small go web app with an endpoint to hit
## Client
A vue.js based js app which is poorly styled but functions.

## up and running
- clone repo
- ` cd server`
- ` go get `
- `cd ../client/`
- `npm install`
- `cd ..`
- `sh startme.sh`
